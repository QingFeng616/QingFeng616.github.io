# Cortex-A7 下的汇编语言学习

## 常用汇编指令

GNU 汇编语法适用于所有的架构，并不是 ARM 独享的。GNU 汇编由一系列的语句组成，每行一条语句，每条语句有三个可选部分：

```asm
label: instruction @ comment
```

- **label**（标号）：表示地址位置，有些指令前面可能会有标号，通过标号可以得到指令的地址。标号也可以用来表示数据地址。注意 `label` 后面的 `:`，任何以 `:` 结尾的标识符都会被识别为一个标号。
- **instruction**（指令）：即汇编指令或伪指令。
- **comment**（注释）：`@` 符号后面的内容为注释，类似 C 语言的 `/* */`。GNU 汇编文件中也可以使用 `/* */` 来注释。

例如：

```asm
add:
    MOVS R0, #0X12   @ 设置 R0 = 0X12
```

上述代码中：`add:` 是标号，`MOVS R0, #0X12` 是指令，`@ 设置 R0 = 0X12` 是注释。

> **注意**：ARM 中的指令、伪指令、伪操作、寄存器名等可以全部使用大写，也可以全部使用小写，但不能大小写混用。

用户可以使用 `.section` 伪操作来定义一个段，汇编系统预定义了以下段名：

| 段名       | 说明               |
| ---------- | ------------------ |
| `.text`    | 代码段             |
| `.data`    | 初始化的数据段     |
| `.bss`     | 未初始化的数据段   |
| `.rodata`  | 只读数据段         |
