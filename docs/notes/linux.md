## 基础命令

### 文件操作

```bash
ls          # 列出文件
cd          # 切换目录
pwd         # 显示当前路径
cp          # 复制文件
mv          # 移动/重命名
rm          # 删除文件
mkdir       # 创建目录
touch       # 创建空文件
cat         # 查看文件内容
```

### 系统信息

```bash
uname -a    # 系统内核信息
top         # 进程监控
df -h       # 磁盘使用情况
free -h     # 内存使用情况
```
